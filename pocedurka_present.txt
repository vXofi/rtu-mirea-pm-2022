Квадратичная сортировка.

1.
Один из простейших алгоритмов сортировки.

На каждом шаге алгоритма проверяются все еще не отсортированным элементы, и,
 как только находится элемент меньше,
 чем первый из неотсортированных, они меняются местами.

2.
алг. сложность.
Для каждого i-го из n элементов происходит n-i сравений. Таким образом,
 общее количество операций даёт вычислительную сложность O(n^2).

3.
устойчивость.

Не является устойчивой, то есть может менять относительный порядок сортируемых элементов.

4.
алгос.

Шаги алгоритма:
	-выбирается очередной элемент списка

	-производится обмен этого значения со значением первой неотсортированной позиции
 	(обмен не нужен, если минимальный элемент уже находится на данной позиции)

	-сортируется хвост списка, исключая из рассмотрения уже отсортированные элементы
